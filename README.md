# Khao Extractor
Khao Extractor is a server-side FabricMC mod that extracts Minecraft data to JSON files.

### Extractors
- [x] Blocks
- [x] Entities
- [x] Items
- [x] Packets
- [x] Recipes
- [x] Enchantments
- [x] Particles
- [x] Biomes
- [x] Villager Professions (Ids, Names)
- [x] Sounds
- [x] Villager Types
- [ ] Translations
